# 三才处境分析 Skill

## 概述

这是一套基于《易经》八卦和三才框架的**伦理处境分析工具**。核心逻辑：

```
三才位置（地/人/天） × 阴阳状态（阳爻/阴爻） × 正负发挥方向（正面/负面）
= 3 × 2 × 2 = 12 个基础概念单元
= 每卦 8 种解读 → 八卦合计 64 种处境
```

**核心前提**：卦象是地形图，不是判决书。吉凶不取决于阴阳本身，而取决于你在每个爻位上的**选择**。

## 用户流程（7 步）

```
用户描述处境
  ↓
Step 1: 选定并缩窄场景          → steps/01-选定场景.md
Step 2: 三才隐喻与场景结合       → steps/02-三才映射.md
Step 3: 阴阳质性映射（中性）     → steps/03-阴阳定性.md
Step 4: 正负面解读（得当/失当）  → steps/04-正负判断.md
Step 5: 生成六爻卦 + 查卦辞     → steps/05-生成六爻卦与卦辞.md
Step 6: 按三才分组解读爻辞      → steps/06-解读爻辞.md
Step 7: 综合建议                → steps/07-综合建议.md
  ↓
输出：处境素描 + 吉凶判断 + 调整建议
```

## 核心原则

1. **阴阳不是好/坏**，是两种质性（向外/向内、刚健/柔顺）
2. **吉凶在时位**，不在阴阳本身
3. **概念永远宽于例子**——先感受隐喻场，再看具体例子
