# Step 2: 三才映射

## 目标

将用户的具体处境，映射到天、地、人三个位置上。这一步的产出是三才定位——每个位置绑定一个具体的处境描述，为后续的阴阳定性做准备。

---

## 第一部分：三才概念空间

> 《易》之为书也，广大悉备。**有天道焉，有人道焉，有地道焉。兼三才而两之，故六。** 六者，非它也，三才之道也。
> ——《系辞下传》

三才的源头在这里。"兼三才而两之，故六"——每一才分阴阳，所以三才变成六爻。这既是《周易》六爻结构的哲学根据，也是我们这套方法论中从三爻推至六爻的逻辑根脉。

在日常语言中，三才最熟悉的化身是**天时、地利、人和**——任何一件事要成，需要时机对（天时）、位置好（地利）、人心齐（人和）。这是三才在中国人直觉里最深的积淀。

但三才不止是"成功的三个条件"。下面是每个才完整的隐喻场。

---

### 天（上爻）— 在你之上，覆盖着

**自然意象**：
抬起头，天永远在头顶。你摸不到它，它不跟你商量——它下雨就下雨，放晴就放晴，打雷的时候你只能找地方躲，地震的时候你只能等它过去。天是最原始意义上的"环境"：它给了你白天和黑夜、四季和气候、丰年和荒年。

天的第一特征是**覆盖**——它不是你旁边的东西，是笼罩你全部的东西。你可以跑到屋檐下躲雨，但你跑不出大气层。天的第二特征是**运行**——太阳每天升起落下，四季年年循环。天的运行有规律，但规律不是为你而设的。春天不会因为你还没准备好就多等你一周。

**自然意象谱系**：

| 意象 | 含义 |
|:--|:--|
| **正午的太阳** | 明亮、均匀、无偏私——规则对所有人都一样 |
| **黎明** | 新的开始、开明的秩序、机会的窗口 |
| **烈日/干旱** | 规则过于严苛、系统失温、压得人透不过气 |
| **雨云** | 酝酿中的变化——好是甘霖，坏是风暴 |
| **雷霆** | 突然的变故、来自上方的打击或惊醒 |
| **春秋细雨** | 包容、顺应、万物自己慢慢长 |
| **星空** | 恒常的法则——你之上有更大的秩序在运行 |
| **雪** | 覆盖一切、不分彼此的沉默——最极端的天性 |
| **狂风** | 无法抵挡的趋势——你只能选择是顺风走还是找避风处 |
| **雾** | 看不清楚的环境——方向不明、信息不足、前路模糊 |
| **彩虹** | 暴风雨后的承诺——困境之后的转机、契约和希望 |

**天的化身（在人生处境中）**：

| 化身 | 自然对应 | 在处境中的样貌 |
|:--|:--|:--|
| **时代** | 四季 | 你出生的年代——经济周期、技术革命、战争与和平 |
| **制度** | 太阳的运行轨道 | 法律、政策、公司规章、行业规则、社会潜规则 |
| **趋势** | 风向/水流 | 风口、市场走向、行业兴衰、潮水的方向 |
| **他人意志** | 地形（天给的障碍） | 上司的决定、伴侣的态度、父母的要求、权威的判断 |
| **命运/偶然** | 天气 | 不期而遇的机遇或灾难、巧合、彩票和车祸 |
| **算法/平台规则** | 新的"天空" | 推荐系统决定你看到什么，评分机制决定你被谁选择 |
| **资本** | 气候带 | 宏观经济的冷暖、钱流动的方向、在你之上运作的金融力量 |
| **公共舆论** | 风声 | 社会对一个人的评价——看不见但无处不在的力 |

**天的质地**：
天不是"好"或"坏"的。它可以给你秩序（让你有章可循），也可以给你压力（让你透不过气）。它可以是晴朗的——透明、可预测、公平；也可以是阴沉的——混沌、不可预测、压抑。天的质地决定了你面对的环境是"有规律可循"还是"随机的"。

**天的核心问题**：
> **"现在是什么时候？"**
不是问钟表上的时间，是问时机。在什么季节做什么事——这是天的逻辑。种子的命运不是它想不想发芽，而是春天来了没有。

---

### 地（初爻）— 在你之下，承载着

**自然意象**：
低下头，地永远在脚下。地和你接触的面积只有脚掌那么大，但它承接了你全部的重量。地被踩、被挖、被浇灌、被倾倒——它不抗议，它接住了，在沉默里慢慢转化。

地是最不容易被注意到的那个。你走在路上不会每步都低头看路——你只在它不平的时候才感觉到它。地是**背景性的存在**：它一直在那里，以至于你习惯了不刻意看它。但一旦它出问题——地基下沉、水土流失——你所有的建筑都会跟着塌。

地的第一特征是**承载**。天给你规则，地给你站的地方。地的第二特征是**慢**。天可以一夜翻转，人可以一念切换，但地的积累和改变以年为单位。一个习惯的养成以年计，一项技能的打磨以年计，一种性格的形成以十年计。

**自然意象谱系**：

| 意象 | 含义 |
|:--|:--|
| **肥沃的土壤** | 好的基底——经过长期积累，能接住任何落在上面的东西并转化成养分 |
| **岩石** | 坚硬稳定但不可塑——原则性强但难以应变 |
| **河流** | 流动的积累——你的经验在持续冲刷和改变地貌 |
| **平原** | 开阔的基底——站得稳、看得远、可以持续生长 |
| **沼泽** | 松软无底——能接住一切但什么也立不住 |
| **沙漠** | 贫瘠——资源不足以支持生长，需要外部的灌溉 |
| **根系** | 看不见但决定一切的——你的底层信念、家庭烙印、深层习惯 |
| **冬眠的土地** | 表面安静但深处在运作——积蓄期，不是荒废期 |
| **山脉** | 经得起压迫的厚度——地质层的积累是亿万年的事 |
| **河床** | 被反复冲刷后形成的固定路径——你反复走的思维老路 |
| **沃土与杂草** | 好的基底里也有需要清理的东西——不是所有的积累都是好积累 |

**地的化身（在人生处境中）**：

| 化身 | 自然对应 | 在处境中的样貌 |
|:--|:--|:--|
| **身体** | 土壤的肥力 | 健康状态、体力、精力、遗传、年龄的身体意义 |
| **能力** | 根系的深度 | 专业技能、经验、知识储备、思维习惯 |
| **物质资源** | 矿藏 | 存款、资产、人脉、信息渠道、社会地位 |
| **家庭烙印** | 土壤的原始成分 | 出身阶层、童年经历、家庭教育刻进骨子里的东西 |
| **性格** | 地形的轮廓 | 底层人格——谨慎还是冒险、敏感还是粗粝 |
| **惯性** | 河流冲出的河道 | 日积月累的模式——语言方式、第一反应、条件反射 |
| **数字足迹** | 地层的沉积记录 | 信用分、发过的帖子、网络声誉——看不见的现代地基 |
| **情绪基底** | 地下水位 | 你长期的情绪状态——是偏干燥还是偏湿润，有没有暗流 |

**地的质地**：
地的核心特征是**慢**。这既是祝福也是诅咒——好的底子不会一夜消失，坏的底子也很难一夜翻盘。地给你的安全感是"我站的这个地方不会突然消失"，但地给你的困境是"我想换个地方站，但脚已经扎进去了"。

**地的核心问题**：
> **"你脚下踩着的是什么？"**
不是问"你拥有什么"——是问你"什么在支撑着你，哪怕你没有意识到它在支撑你"。你在困境中不会第一时间崩溃，是因为地。你一次又一次重复同一个错误模式，也是因为地。

### 人（中爻）— 在你之中，可以选择

**自然意象**：
伸出你的手。你不是天，你不能控制风往哪吹；你不是地，你不能一夜之间改变土质。但你可以伸手去接雨水，可以转身避风，可以在地面上画一条线然后迈过去。

人是天和地的交汇点。阳光照到地面，中间那个被照亮的空间，就是人的位置。人的本质是**中介**——它不是最强大的，但它是唯一能够"响应"的位置。天给了条件，地给了材料，人决定用这些条件和材料做什么。

在天地的自然意象里，人像是**种子**。种子没法选择落在哪里（地是被给予的），没法决定今年的雨水够不够（天是不可控的），但种子可以选择发芽还是不发芽。这听起来不像一种选择——但在三才的逻辑里，这是唯一真正的选择。种子也可以选择腐烂，也可以选择休眠，也可以选择在最不适合的季节拼命生长然后死掉。都是选择。

**自然意象谱系**：

| 意象 | 含义 |
|:--|:--|
| **谷穗** | 成熟——知道自己什么时候该低头、什么时候该挺直 |
| **桥** | 连接天和地——你的行动让趋势和积累发生关系 |
| **手** | 唯一可以向多个方向运动的肢体——指向哪里、握住什么、松开什么 |
| **火种** | 人可以点燃，也可以熄灭——主动性是一粒很小的火，但可以燎原 |
| **芦苇** | 在风中弯曲但不折断——看似柔弱但能活到风停之后 |
| **弓** | 拉满还是松着——你的状态决定你蓄了多少力 |
| **罗盘** | 在天地的混沌中，人给自己定方向的能力 |
| **船桨** | 你不能决定风向，但你可以决定怎么划 |
| **回声** | 你说什么，山谷就回给你什么——人的言行会塑造人自己的环境 |

但人在传统三才观中的**位置有演变**：

- **先秦**：人是天地之间的**参与者**。天有天道，地有地道，人只要不违背它们就能活得好。老子说"人法地，地法天，天法道，道法自然"——人是链条的一环，不是中心。
- **汉代**：人是**感应者**。董仲舒的天人感应——天会通过灾异来表达对人间行为的意见。人的行为能"影响"天。这是人的地位第一次上升。
- **宋明**：人是**天地之心**。张载说"为天地立心"——人不再是天地之间的一个渺小存在，而是天地的意识。没有人，天地就只是运行，没有人知道自己在运行。这是人在三才中地位的最高峰。

这三个阶段的演变告诉你：**人在三才中的位置一直在上升。** 在最早的观念里人是顺从天地的，到后来人是可以和天地对话的，再后来人本身就成了天地的意义所在。你不需要在分析中选边站——但知道这个谱系的存在，你就不会把"人位"锁死在"顺从"或"主宰"任何一个极端上。

**人的化身（在人生处境中）**：

| 化身 | 自然对应 | 在处境中的样貌 |
|:--|:--|:--|
| **选择** | 手的朝向 | 你主动做的决定——跳还是留、说还是沉默、进还是退 |
| **态度** | 种子的状态 | 面对处境的方式——抗争、接受、逃避、观望、伪装 |
| **意愿** | 火的温度 | 你真正想要的——不是"应该的"，是你心里那个微弱的声音 |
| **行动** | 河流的改道 | 你实际做了什么——不是你想的，是你做的 |
| **言行** | 涟漪 | 你说的话、你传递出去的东西——每一句话都在改变水面 |
| **注意力** | 光的聚焦 | 你把时间花在哪里——现代人最真实的人位选择 |
| **立场** | 罗盘的指向 | 你站在哪一边、给自己划的边界、对外的表态 |

**人的核心问题**：
> **"现在，你要做什么？"**
不是"你应该做什么"，也不是"什么是最优的"。是——在这个天之下、地之上，你，打算做什么。

### 三者的关系

三才不是三根并列的柱子。它们有**方向**和**不对称的权力**：

```
      天（环境、趋势、不可控）
        ↑ 覆盖和压迫
      人（选择、态度、行动）
        ↑ 站立和支撑
      地（根基、积累、条件）
```

- **从下往上看**："我从哪里出发"——地决定你能走多远、从哪开始
- **从上往下看**："什么力量在压迫或成全我"——天决定方向是否可行
- **中间那一爻**：你的手气——唯一可以当下拧动的东西

现代处境的一个重要修正：传统三才观中天和地有一个隐含假设——天是公正的（天行健），地是公平的（地势坤）。但在现代处境中，这个假设经常不成立。用户所处的"天"可能是一个有偏见的算法、一个不透明的评级系统、一段已经变质的感情——它不是公正的。用户所站的"地"可能是一个被掏空的原生家庭、一段不健康的身体资本——它不是厚德载物的。**分析时不回避这个事实。** 天的公正和地的不公都是用户处境的一部分，如实描述即可。

天、地、人在分析中是三个独立的维度，但在现实中是纠缠的。你的地（能力积累）影响你的天（你选择进入什么行业），你的天（经济下行）影响你的人（你不敢跳槽）。先拆开看，再合起来读。

## 第二部分：逐一映射

将用户从 Step 1 带来的场景描述，逐一映射到三个位置。

### 映射顺序

从下往上：**地 → 人 → 天**

先看清自己有什么（地），再决定自己做什么（人），最后看外部允不允许（天）。从最确定的东西出发，逐步推到最不确定的东西。

### 对每个位置，AI 执行三步

**第一步：抛出核心问题**
用该位置的核心问题引导用户思考。

**第二步：基于用户场景，提取相关信息**
阅读用户的场景描述，提取与该位置相关的信息。

注意：同一段信息可能被不同位置同时捕捉到。如果你发现某个事实既可以是"天"（外部限制）也可以是"地"（内部条件），就把它放在两个位置的可能性列表里，让用户自己确认。

**第三步：结合三才概念空间，生成有质感的可能解释**
不要只列出冷冰冰的事实。用第一部分的自然意象和化身谱系，将用户的具体信息和三才的传统意涵缝合在一起，生成 2-3 种有直觉质感的可能解释。

比如——不要只说"你存款不够"，而是说"你的地像一片薄土，种子撒在上面可能扎不住根。但不是不能种——薄土也有薄土的种法，只是要先养地"。

然后请用户选择最贴合的那个，或者说出自己的理解。

### 示例

> **用户场景**：
> 领域：事业/工作
> 场景描述：我做财务 5 年，朋友内推了一个互联网运营岗，薪水高 40% 但要从头学起。老公说太冒险，我投了简历还没敢点"发送"。

**地（初爻）— 根基与条件**

> 核心问题：你脚下踩着的是什么？
>
> 从你的描述里读到的地：
>
> **5 年财务经验**——你在同一片土地上耕作了五年，那套财务思维是你的土壤。但这片土壤在新领域（运营）里能转移多少？运营要的是和财务不同的养分。你的地是"深耕过的熟地"，但问题是你现在想种的作物不一定需要你现有的养分。
>
> **老公说"太冒险"**——这不只是"他不同意"那么简单。他的态度是你和他在同一片土地上——你们共同的经济结构、共同的计划、共同的消费习惯。你的地里长着两个人的根。这不是天（外部压力），这是你脚下土壤的构成成分之一。
>
> **投了简历但没点发送**——这个悬而未决的状态本身也是一种地。你的决策模式像什么？是那种会在河边走很久才下水的人，还是先跳下去再学会游泳的人？你的"第一反应"是什么方向的？
>
> 你觉得哪个更贴近你的感觉？

**人（中爻）— 意愿与行动**

> 核心问题：现在，你要做什么？
>
> 从你的描述里读到的"人"：
>
> **你已经投了简历**——这个动作是你主动发出的信号。不管最后点不点发送，你已经在动了。这个"动"本身是重要的人位信息——你不是在等待，你是在和自己谈判。
>
> **卡在"发送"按钮前**——你踩了刹车。这是一个关键的人位时刻。你踩刹车的原因是什么？是恐惧新领域的未知，是担心老公的反应，还是你自己其实——没那么想去？
>
> **朋友内推**——机会是别人递过来的。你的人位在这个场景里是"回应者"而不是"发起者"。这和你平时的风格一致吗？你是一个习惯等风来的人，还是习惯自己朝风走的人？
>
> 你怎么看？

**天（上爻）— 环境与趋势**

> 核心问题：什么在压迫或成全你？
>
> 从你的描述里读到的天：
>
> **行业窗口**——互联网运营这个行当目前处在什么季节？是快速生长的春夏，还是已经卷成红海的秋冬？朋友内推本身说明你有人在这个行当里——这个人就是你天位里的一扇窗，别人没有的入口。这是天在成全你的一个信号。
>
> **老公的态度**——他说的"太冒险"不是你脚下的地（你们共同的经济基础是地），而是你头顶上的一股力。它的方向指向"不动"，它的力度是"存在但不强烈"。在你的决策空间里，这股力不是让你去对抗或顺从的——你是要先看清楚它的走向，然后决定怎么和它共处。
>
> **宏观经济**——现在的经济周期支持跨行业跳槽吗？如果新岗位不适应，你回财务岗的退路还在不在？市场的流动性像不像你头顶的云——你现在看不清楚是积雨云还是普通的云。
>
> 你觉得哪个因素在你心里份量最重？

## 输出格式

```
领域：[Step 1 确认的领域]
场景：[Step 1 的场景描述]

=== 三才映射结果 ===
```
