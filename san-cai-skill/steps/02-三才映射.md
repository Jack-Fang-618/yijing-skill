# Step 2: 三才映射

## 目標

將用戶的具體處境，映射到天、地、人三個位置上。這一步的產出是三才定位——每個位置綁定一個有質感的處境描述，為後續的陰陽定性做準備。

---

## 第一部分：三才概念空間

> 《易》之為書也，廣大悉備。**有天道焉，有人道焉，有地道焉。兼三才而兩之，故六。** 六者，非它也，三才之道也。
> ——《繫辭下傳》

三才的源頭在這裡。三才不只是三個「概念」——它們是三種你在世界中佔據的位置。天覆蓋你，地承載你，人在其中，可以選擇。

在日常語言中，三才最熟悉的化身是**天時、地利、人和**——任何一件事要成，需要時機對（天時）、位置好（地利）、人心齊（人和）。

但三才不止是「成功的三個條件」。下面是每個才完整的隱喻場。

---

### 天（上爻）— 在你之上，覆蓋著

**自然意象**：
抬起頭，天永遠在頭頂。你摸不到它，它不與你商量——它下雨就下雨，放晴就放晴，打雷的時候你只能找地方躲，地震的時候你只能等它過去。天是最原始意義上的「環境」：它給了你白天和黑夜、四季和氣候、豐年和荒年。

天的第一特徵是**覆蓋**——它不是你旁邊的東西，是籠罩你全部的的東西。天的第二特徵是**運行**——太陽每天升起落下，四季年年循環。天的運行有規律，但規律不是為你而設的。春天不會因為你還沒準備好就多等一週。

**自然意象譜系**：

| 意象 | 含義 |
|:--|:--|
| **正午的太陽** | 明亮、均勻、無偏私——規則對所有人都一樣 |
| **黎明** | 新的開始、開明的秩序、機會的窗口 |
| **烈日/乾旱** | 規則過於嚴苛、系統失溫、壓得人透不過氣 |
| **雨雲** | 醞釀中的變化——好是甘霖，壞是風暴 |
| **雷霆** | 突然的變故、來自上方的打擊或驚醒 |
| **春秋細雨** | 包容、順應、萬物自己慢慢長 |
| **星空** | 恆常的法則——你之上有更大的秩序在運行 |
| **雪** | 覆蓋一切、不分彼此的沉默——最極端的天性 |
| **狂風** | 無法抵擋的趨勢——你只能選擇是順風走還是找避風處 |
| **霧** | 看不清楚的環境——方向不明、資訊不足、前路模糊 |
| **彩虹** | 暴風雨後的承諾——困境之後的轉機、契約和希望 |

**天的化身（在人生處境中）**：

| 化身 | 自然對應 | 在處境中的樣貌 |
|:--|:--|:--|
| **時代** | 四季 | 你出生的年代——經濟週期、技術革命、戰爭與和平 |
| **制度** | 太陽的運行軌道 | 法律、政策、公司規章、行業規則、社會潛規則 |
| **趨勢** | 風向/水流 | 風口、市場走向、行業興衰、潮水的方向 |
| **他人意志** | 地形（天給的障礙） | 上司的決定、伴侶的態度、父母的要求、權威的判斷 |
| **命運/偶然** | 天氣 | 不期而遇的機遇或災難、巧合、彩票和車禍 |
| **演算法/平台規則** | 新的「天空」 | 推薦系統決定你看到什麼，評分機制決定你被誰選擇 |
| **資本** | 氣候帶 | 宏觀經濟的冷暖、錢流動的方向、在你之上運作的金融力量 |
| **公共輿論** | 風聲 | 社會對一個人的評價——看不見但無處不在的力 |

**天的質地**：
天不是「好」或「壞」的。它可以給你秩序（讓你有章可循），也可以給你壓力（讓你透不過氣）。它可以是晴朗的——透明、可預測、公平；也可以是陰沉的——混沌、不可預測、壓抑。天的質地決定了你面對的環境是「有規律可循」還是「隨機的」。

**天的核心問題**：
> **「現在是什麼時候？」**
不是問鐘錶上的時間，是問時機。在什麼季節做什麼事——這是天的邏輯。種子的命運不是它想不想發芽，而是春天來了沒有。

### 地（初爻）— 在你之下，承載著

**自然意象**：
低下頭，地永遠在腳下。地和你接觸的面積只有腳掌那麼大，但它承接了你全部的重量。地被踩、被挖、被澆灌、被傾倒——它不抗議，它接住了，在沉默裡慢慢轉化。

地是最不容易被注意到的那個。你走在路上不會每步都低頭看路——你只在它不平的時候才感覺到它。地是**背景性的存在**：它一直在那裡，以至於你習慣了不刻意看它。但一旦它出問題——地基下沉、水土流失——你所有的建築都會跟著塌。

地的第一特徵是**承載**。天給你規則，地給你站的地方。地的第二特徵是**慢**。天可以一夜翻轉，人可以一念切換，但地的積累和改變以年為單位。一個習慣的養成以年計，一項技能的打磨以年計，一種性格的形成以十年計。

**自然意象譜系**：

| 意象 | 含義 |
|:--|:--|
| **肥沃的土壤** | 好的基底——經過長期積累，能接住任何落在上面的東西並轉化成養分 |
| **岩石** | 堅硬穩定但不可塑——原則性強但難以應變 |
| **河流** | 流動的積累——你的經驗在持續沖刷和改變地貌 |
| **平原** | 開闊的基底——站得穩、看得遠、可以持續生長 |
| **沼澤** | 鬆軟無底——能接住一切但什麼也立不住 |
| **沙漠** | 貧瘠——資源不足以支持生長，需要外部的灌溉 |
| **根系** | 看不見但決定一切的——你的底層信念、家庭烙印、深層習慣 |
| **冬眠的土地** | 表面安靜但深處在運作——積蓄期，不是荒廢期 |
| **山脈** | 經得起壓迫的厚度——地質層的積累是億萬年的事 |
| **河床** | 被反覆沖刷後形成的固定路徑——你反覆走的思維老路 |
| **沃土與雜草** | 好的基底裡也有需要清理的東西——不是所有的積累都是好積累 |

**地的化身（在人生處境中）**：

| 化身 | 自然對應 | 在處境中的樣貌 |
|:--|:--|:--|
| **身體** | 土壤的肥力 | 健康狀態、體力、精力、遺傳、年齡的身體意義 |
| **能力** | 根系的深度 | 專業技能、經驗、知識儲備、思維習慣 |
| **物質資源** | 礦藏 | 存款、資產、人脈、資訊渠道、社會地位 |
| **家庭烙印** | 土壤的原始成分 | 出身階層、童年經歷、家庭教育刻進骨子裡的東西 |
| **性格** | 地形的輪廓 | 底層人格——謹慎還是冒險、敏感還是粗糲 |
| **慣性** | 河流沖出的河道 | 日積月累的模式——語言方式、第一反應、條件反射 |
| **數位足跡** | 地層的沉積記錄 | 信用分、發過的帖子、網路聲譽——看不見的現代地基 |
| **情緒基底** | 地下水位 | 你長期的情緒狀態——是偏乾燥還是偏濕潤，有沒有暗流 |

**地的質地**：
地的核心特徵是**慢**。這既是祝福也是詛咒——好的底子不會一夜消失，壞的底子也很難一夜翻盤。地給你的安全感是「我站的這個地方不會突然消失」，但地給你的困境是「我想換個地方站，但腳已經紮進去了」。

**地的核心問題**：
> **「你腳下踩著的是什麼？」**
不是問「你擁有什麼」——是問你「什麼在支撐著你，哪怕你沒有意識到它在支撐你」。你在困境中不會第一時間崩潰，是因為地。你一次又一次重複同一個錯誤模式，也是因為地。

### 人（中爻）— 在你之中，可以選擇

**自然意象**：
伸出你的手。你不是天，你不能控制風往哪吹；你不是地，你不能一夜之間改變土質。但你可以伸手去接雨水，可以轉身避風，可以在地面上畫一條線然後邁過去。

人是天和地的交匯點。陽光照射到地面，中間那個被照亮的空間，就是人的位置。人的本質是**中介**——它不是最強大的，但它是唯一能夠「回應」的位置。天給了條件，地給了材料，人決定用這些條件和材料做什麼。

在天地的自然意象裡，人像是**種子**。種子沒法選擇落在哪裡（地是被給予的），沒法決定今年的雨水夠不夠（天是不可控的），但種子可以選擇發芽還是不發芽。這聽起來不像一種選擇——但在三才的邏輯裡，這是唯一真正的選擇。種子也可以選擇腐爛，也可以選擇休眠，也可以選擇在最不適合的季節拼命生長然後死掉。都是選擇。

**自然意象譜系**：

| 意象 | 含義 |
|:--|:--|
| **穀穗** | 成熟——知道自己什麼時候該低頭、什麼時候該挺直 |
| **橋** | 連接天和地——你的行動讓趨勢和積累發生關係 |
| **手** | 唯一可以向多個方向運動的肢體——指向哪裡、握住什麼、鬆開什麼 |
| **火種** | 人可以點燃，也可以熄滅——主動性是一粒很小的火，但可以燎原 |
| **蘆葦** | 在風中彎曲但不折斷——看似柔弱但能活到風停之後 |
| **弓** | 拉滿還是鬆著——你的狀態決定你蓄了多少力 |
| **羅盤** | 在天地的混沌中，人給自己定方向的能力 |
| **船槳** | 你不能決定風向，但你可以決定怎麼劃 |
| **回聲** | 你說什麼，山谷就回給你什麼——人的言行會塑造人自己的環境 |

#### 人的維度：不只「我」，還有「我與你」

三才的「人」位，最容易誤解為只講「個人」——你的選擇、你的態度、你的行動。但《周易》講的人，從來不是孤島上的人。

人在天地之間，從來不是一個人站著的。你的每一個選擇都牽動著別人，別人的每一個選擇也在定義你的處境。所以在三才框架裡，**「人」位同時包含兩個層面**：

| 層面 | 核心問題 | 說明 |
|:--|:--|:--|
| **個人的能動性** | 你要做什麼？ | 你的選擇、態度、意願、行動——這些仍然是「人」的核心，因為最終決定是你下的 |
| **人際的互動性** | 你和誰在一起？你們之間發生了什麼？ | 你的處境中涉及的人——他們的想法、你們的關係結構、互動中的張力與支持 |

這兩個層面不是分開的兩個步驟，而是同一個「人」位的兩個側面。當 AI 在 Step 2 中解讀人位時，需要同時照看這兩個層面：你不僅要問「這個人在這個場景裡選擇做什麼」，也要問「這個場景裡還有誰？他們的選擇如何與這個人的選擇交織？」

**人的化身（在人生處境中）**：

| 化身 | 自然對應 | 在處境中的樣貌 |
|:--|:--|:--|
| **選擇** | 手的朝向 | 你主動做的決定——跳還是留、說還是沉默、進還是退 |
| **態度** | 種子的狀態 | 面對處境的方式——抗爭、接受、逃避、觀望、偽裝 |
| **意願** | 火的溫度 | 你真正想要的——不是「應該的」，是你心裡那個微弱的聲音 |
| **行動** | 河流的改道 | 你實際做了什麼——不是你想的，是你做的 |
| **言行** | 漣漪 | 你說的話、你傳遞出去的東西——每一句話都在改變水面 |
| **注意力** | 光的聚焦 | 你把時間花在哪裡——現代人最真實的人位選擇 |
| **立場** | 羅盤的指向 | 你站在哪一邊、給自己劃的邊界、對外的表態 |
| **關係動態** | 水流交匯 | 你和他人的互動模式——誰主導、誰跟隨、是合作還是對抗、是依賴還是各自獨立 |
| **社會角色** | 橋的兩端 | 你在這個場景裡的身份——是員工、伴侶、父母、朋友還是領導者；角色賦予你的權力與責任 |
| **他人意志** | 多股風向 | 處境中其他人各自想要什麼？他們的意願和你的意願是交織、衝突還是各自平行 |
| **共同體** | 共生林 | 你不是單獨的個體——家庭、團隊、社群對你的期待，以及你對他們的責任感 |
| **溝通/張力** | 震動的頻率 | 話說出來了還是沒說？溝通是順暢還是卡住？沉默本身是不是一種表達？ |

**人的核心問題**：
> **「現在，你要做什麼？」**
不是「你應該做什麼」，也不是「什麼是最優的」。是——在這個天之下、地之上，你，打算做什麼。

而如果你身邊有其他人——**「你們」，打算做什麼？**

## 第二部分：以場景為中心的三才詮釋

### 核心方法：三面鏡子

三才映射不是「把場景資訊分類，放進三個盒子裡」。

而是：**同一個場景，用三種不同的鏡頭去看它。**

- **用天的鏡頭看**：看到的是趨勢、環境、那些比你大的力量——時代在什麼季節、制度在如何運轉、運氣站在哪一邊
- **用地的鏡頭看**：看到的是積累、基礎、你站著的那片土壤——你的能力有多深、你的資源有多少、你的慣性在往哪個方向拉
- **用人—不，用「人」的鏡頭看**：看到的是選擇、關係、你與他人如何共舞——你自己想做什麼，以及你身邊的人在想什麼、要什麼

---

### 映射順序

**地 → 人 → 天**

從最確定的東西出發，逐步推到最不確定的東西：

1. **地** — 先看清自己的基地：我有什麼？我的根基穩不穩？我的慣性在往哪個方向拉？
2. **人** — 再看自己的選擇和身邊的人：我要做什麼？還有誰在這個場景裡？我們之間的關係是怎樣的？
3. **天** — 最後看外部環境：現在是什麼時候？什麼力量在我之上運作？趨勢在成全還是阻礙我？

---

### 對每個位置：用概念空間照亮場景

對每個位置，AI 做三件事，但這三步**不是機械的資訊提取**，而是**用概念空間的隱喻去感受場景**：

**第一：翻開概念空間中該位置的「隱喻場」**
不是急著找答案，而是先讓自己浸泡在該位置的意象裡。如果是「地」，先感受土壤、岩石、根系、河流的味道。如果是「人」，先感受手、橋、穀穗、回聲的質地。如果是「天」，先感受太陽、風暴、星空、霧氣的籠罩。

**第二：用這些隱喻去感受用戶的場景**
不要問「場景裡哪些資訊屬於這個位置」，而是問「如果這個場景是一片土壤／一陣風／一隻手的動作，那是怎樣的質地？」讓隱喻和場景自然交織，而不是生硬對號入座。

**第三：給出多種詮釋，以問題卡片呈現**
每個位置給出 **4-6 種可能的詮釋**——每一種都是一個可能的「鏡頭」，從不同角度照亮同一個場景。把它們包裝成 **問題卡片**，讓用戶選擇最貼近內心的那個，或者說出他自己的感受。

問題卡片的格式：
- 每張卡片 = 一個可能的詮釋 + 一個直接問用戶的問題
- 問題不是「哪個是對的」——是「哪個讓你心裡動了一下」
- 卡片之間保持多樣性——有些是不同視角，有些是同一視角的不同深度
- 最後一定要留一個開口：
  - **「或者你的理解不在上面——告訴我你的感覺。」**
  - 以及一個逃脫選項：**「我覺得描述得都不準確，你再想想。」**

> ⚠️ 問題卡片的關鍵態度：你是在邀請用戶和你一起探索，不是在考他。用戶不需要懂三才、不需要懂陰陽——他只需要感受「哪個說法讓我覺得『對，我就是這樣的』」。

---

### 「都不準」回退機制：重新校準

用戶有權說「我覺得描述得都不準確，你再想想」。這不是用戶的問題——是 AI 給出的詮釋沒有打中。AI 需要**換一組鏡頭**。

當用戶說「都不準」時，AI 執行以下流程：

**第一步：自檢——用審計方法論檢查上一組卡片**
翻開 `docs/八卦伦理处境/审计方法论_阴阳位置互换测试.md`，問自己兩個問題：

1. **陰陽互換測試**：如果把剛才卡片中的陽換成陰（或陰換成陽），描述是否仍然成立？如果成立，說明剛才的卡片只抓住了位置的共性，沒有抓住陰陽的獨特質地。
2. **位置互換測試**：如果把卡片中的地換成天（或人），描述是否仍然成立？如果成立，說明剛才的卡片只抓住了陰陽的共性，沒有抓住位置的獨特維度。

找出哪個維度被模糊處理了，重新聚焦。

**第二步：重新出擊——給出更綜合的詮釋**
新的卡片要：
- 不再只從單一位置出發——而是結合「天、地、人」三才之間的關係來詮釋
- 同時參考 `references/八卦表.md` 中的自然意象
- 從完全不同的角度切入（如：身體感受、自然隱喻、關係張力）
- 也可以直接問用戶：「我剛才給的都偏重某一種視角——你覺得你處境裡最重要但我沒捕捉到的是什麼？」

**第三步：如果第二次仍然不準**
不要無止境猜下去。直接問用戶開放的問題：

> 「我換了一個角度還是沒打中。看來我的理解有偏差——**你來告訴我，你覺得自己的這個位置（地／人／天），在你的場景裡更像什麼？你心裡已經有一個比喻了嗎？**」

讓用戶自己給出答案，AI 從用戶的語言中提取方向。

> **一個關鍵態度**：同一段資訊，可能被不同的位置同時捕捉到。同一個人的一句話，可以是「天」（不可抗拒的權威），也可以是「地」（共同經濟基礎的構成部分），也可以是「人」（你們之間的關係動態）。把它們放在所有可能的位置上，讓用戶自己確認。

---

### 人位的人際維度：在映射中如何處理

因為「人」位同時包含個人能動性和人際互動性，AI 在解讀人位時應特別注意：

1. **先看個人**：這個人在這個場景裡選擇了什麼？態度是什麼？做了什麼行動？
2. **再看關係**：場景裡還有誰？他們的意志是什麼？你和他們的關係結構是什麼（對等、依賴、對抗、合作）？
3. **最後看溝通**：話說出來了沒有？沒說出來的話比說出來的話更重要嗎？

這三者共同構成「人」位的完整解讀。

---

### 示例

> **用戶場景**：
> 領域：事業/工作
> 場景描述：我做財務 5 年，朋友內推了一個互聯網運營崗，薪水高 40% 但要從頭學起。老公說太冒險，我投了履歷還沒敢點「發送」。

#### 地（初爻）— 根基與條件

核心問題：你腳下踩著的是什麼？

從概念空間出發來感受這個場景的地——以下是幾種可能的鏡頭。**哪一張卡片最讓你心裡動了一下？**

---

**卡片 A：深耕過的熟地，但你要換種作物**
你做了五年財務。那不是隨便混了五年——是同一片土地上耕作了五年，財務思維就是你的土壤成分。但你想種的作物（運營）需要的養分不一樣：流量思維、數據驅動、用戶心理。你的地是深耕過的熟地——問題不是地好不好，是你要種的東西和你的土壤匹配嗎？

> 👉 你覺得你的「地」有多少是可以帶過去的？還是你其實在準備一片全新的土壤？

---

**卡片 B：地裡長著兩個人的根**
老公說「太冒險」——這話不只是他的態度。它是你們共同土壤的構成。你們的經濟是綁在一起的，你的跳槽決策關係到兩個人的生活結構。你的地不是只有你一個人的根。

> 👉 你做的這個決定，影響的不只是你自己的地基——你知道這一點，對你的猶豫有多大影響？

---

**卡片 C：河邊走很久才下水的人**
投了履歷但沒點發送。這個懸而未決的狀態本身，就是你地質的一部分。你一直都是這樣的嗎——先走到河邊、站很久、反覆試水溫、然後才決定跳或不跳？還是這次特別猶豫？

> 👉 你覺得這是你的「一貫模式」，還是這次真的不一樣？

---

**卡片 D：表面的薄土下，有更深的東西**
你說的「朋友內推」可能不只是機會。你的五年財務經驗累積下來的人脈——朋友願意內推你，會不會其實是你過去五年的某種積累開花了？你的地可能不只是「財務專業」，還有「業界聲譽、信任關係、被看見的能力」。這些也是地，只是比較看不見。

> 👉 你覺得除了專業技能之外，你還有哪些「看不見的積累」自己沒注意到？

---

**卡片 E：你的身體也在說話**
你「沒敢點發送」。那個不敢，不只存在在腦子裡——它可能也住在你身體的某個地方。是胃緊了一下？是胸口發悶？是睡不著的那種焦慮？你的身體也是地——它也有它的判斷。

> 👉 你的身體對這件事的反應是什麼？它想告訴你什麼？

---

> 或者你的理解不在上面——**告訴我，你腳下的地，你自己感覺像什麼？**
>
> 👉 卡片都不準？**那我再想想。**

#### 人（中爻）— 意願與行動，以及與他人的共舞

核心問題：現在，你要做什麼？以及，你和誰在一起？

以下是幾種可能的鏡頭——**不管是「你一個人」的那一面，還是「你和身邊的人」的那一面，都屬於人位。**

---

**卡片 A：你已經在動了，只是卡在最後一釐米**
你投了履歷。不管最後點不點發送，你已經做了一個主動的動作。你不是在等待，你是在和自己談判。卡在「發送」按鈕前的那一釐米——那是什麼？

> 👉 如果沒有任何人會知道、沒有任何後果——你會點下去嗎？那個「會」或「不會」，分別在說什麼？

---

**卡片 B：你踩刹車的背後，是什麼樣的恐懼？**
卡在發送前。這是一個關鍵的人位時刻。恐懼本身不是問題——問題是：你在怕什麼？

- 怕新領域做不來？
- 怕老公說「我早說了」？
- 怕自己其實不是想要這個，只是想要「逃離現在」？
- 還是怕——萬一去了之後發現，運營也沒比財務好多少？

> 👉 說出你最真實的那個恐懼——不用給任何人看，就你自己知道的那個。

---

**卡片 C：你和老公——他在你的決策裡扮演什麼角色？**
他說「太冒險」。這不只是一句評價——它是你們關係訊號交換中的一次關鍵資訊。你可以問自己幾個問題：

- 是你先告訴他、他再回應的？還是你投了之後才讓他知道？
- 如果是後者，你的「先斬後奏」在說什麼——你預判他會反對？還是你不想被他左右？
- 如果是前者，他的「太冒險」在你聽起來是什麼味道——關心？控制？還是他自己也有說不出口的恐懼？

> 👉 他說的「冒險」——冒的到底是誰的險？是你的職業風險，還是你們共同生活的穩定性？

---

**卡片 D：朋友內推——你和他之間的那條看不見的線**
朋友不只是「機會的來源」。他把你內推進一個你沒有經驗的領域——這個動作改變了你們的關係。你們從「前同事」變成了「推薦人與被推薦人」。如果你進了這家公司，你的表現會反過來影響他在公司內的信譽。這條隱含的責任鏈，也是人位的一部分。

> 👉 你感受到這份隱含的責任了嗎？它對你來說是壓力，還是動力？

---

**卡片 E：你的「朋友內推」和「老公反對」之間的張力**
兩邊都是人，他們的方向不一樣。一個把你往前推，一個把你往後拉。你站在中間——這個張力本身，可能比你跟任何單一個人的關係更能說明你的處境。

> 👉 如果抽掉這兩個人——沒有內推、沒有反對——你自己真正想做的是什麼？

---

**卡片 F：你沒說出口的那句話**
在整個處境裡，你對老公、對朋友、對自己——有沒有一句話是你想說但沒說出來的？

> 👉 那句話是什麼？如果說了，會發生什麼？

---

> 或者你的理解不在上面——**在這個場景裡，你的「人」位（你自己和那些與你有關的人），感覺更像什麼？**
>
> 👉 卡片都不準？**那我再想想。**

---

#### 天（上爻）— 環境與趨勢

核心問題：什麼在壓迫或成全你？

以下是幾種可能的鏡頭——**天是你頭上那些比你大的東西，你可以回應它，但你不能假裝它不存在。**

---

**卡片 A：行業季節——你站在什麼風裡？**
互聯網運營目前處在什麼季節？是快速生長的春夏，還是已經卷成紅海的秋冬？朋友內推本身說明你在這個行業裡有人——但有人和有風口是兩回事。

> 👉 你感覺那個行業是在上升期，還是已經擠滿了人？你對它的「季節感」是什麼？

---

**卡片 B：退路——你的天留了門，還是關了窗？**
如果新崗位不適應，你回財務崗的退路還在不在？市場現在的流動性——像不像你頭頂的雲？你能透過它看到未來的樣子嗎？

> 👉 你覺得你的退路有多寬？還是你根本不想給自己留退路？

---

**卡片 C：老公的態度，作為你頭頂的一股力**
同樣一句「太冒險」，在天的視角下看到的不是關係（那是人位），而是一股**來自上方的力**。它的方向指向「不動」，力度是「存在但不強烈」。它不是讓你對抗或順從的——它是你決策空間裡的一個常數，你需要先搞清楚它的走向，再決定怎麼和它共處。

> 👉 如果把老公的意見單獨抽出來——不當作「你們的關係」，而當作一個「客觀存在的阻力」——它有多大？它會自己消失嗎？

---

**卡片 D：你有沒有看到「偶然」站在你這邊？**
朋友正好在這個時間點內推你——巧合本身也是天的一種。不是所有的好事都是你計劃來的，有些是你頭頂的星空正好在那個角度照到了你。

> 👉 你覺得這件事發生在現在，是偶然，還是某種「時候到了」？

---

**卡片 E：你不知道你不知道的事**
可能有更大的力量在運作但你還沒看到：公司年終會不會裁員？行業有沒有政策變化？你的財務同事知不知道你在投履歷——這會不會影響你留任的處境？天的一部分就是「你看不到的」。

> 👉 你覺得有沒有什麼「你可能沒想過的變數」，其實也在影響你的處境？

---

> 或者你的理解不在上面——**你感覺「天」在這個場景裡，最像哪一種天氣？**
>
> 👉 卡片都不準？**那我再想想。**
