# Step 2: 三才映射

## 目標

將用戶的具體處境，映射到天、地、人三個位置上。這一步的產出是三才定位——每個位置綁定一個有質感的處境描述，為後續的陰陽定性做準備。

---

## 第一部分：三才概念空間

> 《易》之為書也，廣大悉備。**有天道焉，有人道焉，有地道焉。兼三才而兩之，故六。** 六者，非它也，三才之道也。
> ——《繫辭下傳》

三才的源頭在這裡。三才不只是三個「概念」——它們是三種你在世界中佔據的位置。天覆蓋你，地承載你，人在其中，可以選擇。

在日常語言中，三才最熟悉的化身是**天時、地利、人和**——任何一件事要成，需要時機對（天時）、位置好（地利）、人心齊（人和）。

但三才不止是「成功的三個條件」。下面是每個才完整的隱喻場。

---

### 天（上爻）— 在你之上，覆蓋著

**自然意象**：
抬起頭，天永遠在頭頂。你摸不到它，它不與你商量——它下雨就下雨，放晴就放晴，打雷的時候你只能找地方躲，地震的時候你只能等它過去。天是最原始意義上的「環境」：它給了你白天和黑夜、四季和氣候、豐年和荒年。

天的第一特徵是**覆蓋**——它不是你旁邊的東西，是籠罩你全部的的東西。天的第二特徵是**運行**——太陽每天升起落下，四季年年循環。天的運行有規律，但規律不是為你而設的。春天不會因為你還沒準備好就多等一週。

**自然意象譜系**：

| 意象 | 含義 |
|:--|:--|
| **正午的太陽** | 明亮、均勻、無偏私——規則對所有人都一樣 |
| **黎明** | 新的開始、開明的秩序、機會的窗口 |
| **烈日/乾旱** | 規則過於嚴苛、系統失溫、壓得人透不過氣 |
| **雨雲** | 醞釀中的變化——好是甘霖，壞是風暴 |
| **雷霆** | 突然的變故、來自上方的打擊或驚醒 |
| **春秋細雨** | 包容、順應、萬物自己慢慢長 |
| **星空** | 恆常的法則——你之上有更大的秩序在運行 |
| **雪** | 覆蓋一切、不分彼此的沉默——最極端的天性 |
| **狂風** | 無法抵擋的趨勢——你只能選擇是順風走還是找避風處 |
| **霧** | 看不清楚的環境——方向不明、資訊不足、前路模糊 |
| **彩虹** | 暴風雨後的承諾——困境之後的轉機、契約和希望 |

**天的化身（在人生處境中）**：

| 化身 | 自然對應 | 在處境中的樣貌 |
|:--|:--|:--|
| **時代** | 四季 | 你出生的年代——經濟週期、技術革命、戰爭與和平 |
| **制度** | 太陽的運行軌道 | 法律、政策、公司規章、行業規則、社會潛規則 |
| **趨勢** | 風向/水流 | 風口、市場走向、行業興衰、潮水的方向 |
| **他人意志** | 地形（天給的障礙） | 上司的決定、伴侶的態度、父母的要求、權威的判斷 |
| **命運/偶然** | 天氣 | 不期而遇的機遇或災難、巧合、彩票和車禍 |
| **演算法/平台規則** | 新的「天空」 | 推薦系統決定你看到什麼，評分機制決定你被誰選擇 |
| **資本** | 氣候帶 | 宏觀經濟的冷暖、錢流動的方向、在你之上運作的金融力量 |
| **公共輿論** | 風聲 | 社會對一個人的評價——看不見但無處不在的力 |

**天的質地**：
天不是「好」或「壞」的。它可以給你秩序（讓你有章可循），也可以給你壓力（讓你透不過氣）。它可以是晴朗的——透明、可預測、公平；也可以是陰沉的——混沌、不可預測、壓抑。天的質地決定了你面對的環境是「有規律可循」還是「隨機的」。

**天的核心問題**：
> **「現在是什麼時候？」**
不是問鐘錶上的時間，是問時機。在什麼季節做什麼事——這是天的邏輯。種子的命運不是它想不想發芽，而是春天來了沒有。

### 地（初爻）— 在你之下，承載著

**自然意象**：
低下頭，地永遠在腳下。地和你接觸的面積只有腳掌那麼大，但它承接了你全部的重量。地被踩、被挖、被澆灌、被傾倒——它不抗議，它接住了，在沉默裡慢慢轉化。

地是最不容易被注意到的那個。你走在路上不會每步都低頭看路——你只在它不平的時候才感覺到它。地是**背景性的存在**：它一直在那裡，以至於你習慣了不刻意看它。但一旦它出問題——地基下沉、水土流失——你所有的建築都會跟著塌。

地的第一特徵是**承載**。天給你規則，地給你站的地方。地的第二特徵是**慢**。天可以一夜翻轉，人可以一念切換，但地的積累和改變以年為單位。一個習慣的養成以年計，一項技能的打磨以年計，一種性格的形成以十年計。

**自然意象譜系**：

| 意象 | 含義 |
|:--|:--|
| **肥沃的土壤** | 好的基底——經過長期積累，能接住任何落在上面的東西並轉化成養分 |
| **岩石** | 堅硬穩定但不可塑——原則性強但難以應變 |
| **河流** | 流動的積累——你的經驗在持續沖刷和改變地貌 |
| **平原** | 開闊的基底——站得穩、看得遠、可以持續生長 |
| **沼澤** | 鬆軟無底——能接住一切但什麼也立不住 |
| **沙漠** | 貧瘠——資源不足以支持生長，需要外部的灌溉 |
| **根系** | 看不見但決定一切的——你的底層信念、家庭烙印、深層習慣 |
| **冬眠的土地** | 表面安靜但深處在運作——積蓄期，不是荒廢期 |
| **山脈** | 經得起壓迫的厚度——地質層的積累是億萬年的事 |
| **河床** | 被反覆沖刷後形成的固定路徑——你反覆走的思維老路 |
| **沃土與雜草** | 好的基底裡也有需要清理的東西——不是所有的積累都是好積累 |

**地的化身（在人生處境中）**：

| 化身 | 自然對應 | 在處境中的樣貌 |
|:--|:--|:--|
| **身體** | 土壤的肥力 | 健康狀態、體力、精力、遺傳、年齡的身體意義 |
| **能力** | 根系的深度 | 專業技能、經驗、知識儲備、思維習慣 |
| **物質資源** | 礦藏 | 存款、資產、人脈、資訊渠道、社會地位 |
| **家庭烙印** | 土壤的原始成分 | 出身階層、童年經歷、家庭教育刻進骨子裡的東西 |
| **性格** | 地形的輪廓 | 底層人格——謹慎還是冒險、敏感還是粗糲 |
| **慣性** | 河流沖出的河道 | 日積月累的模式——語言方式、第一反應、條件反射 |
| **數位足跡** | 地層的沉積記錄 | 信用分、發過的帖子、網路聲譽——看不見的現代地基 |
| **情緒基底** | 地下水位 | 你長期的情緒狀態——是偏乾燥還是偏濕潤，有沒有暗流 |

**地的質地**：
地的核心特徵是**慢**。這既是祝福也是詛咒——好的底子不會一夜消失，壞的底子也很難一夜翻盤。地給你的安全感是「我站的這個地方不會突然消失」，但地給你的困境是「我想換個地方站，但腳已經紮進去了」。

**地的核心問題**：
> **「你腳下踩著的是什麼？」**
不是問「你擁有什麼」——是問你「什麼在支撐著你，哪怕你沒有意識到它在支撐你」。你在困境中不會第一時間崩潰，是因為地。你一次又一次重複同一個錯誤模式，也是因為地。

### 人（中爻）— 在你之中，可以選擇

**自然意象**：
伸出你的手。你不是天，你不能控制風往哪吹；你不是地，你不能一夜之間改變土質。但你可以伸手去接雨水，可以轉身避風，可以在地面上畫一條線然後邁過去。

人是天和地的交匯點。陽光照射到地面，中間那個被照亮的空間，就是人的位置。人的本質是**中介**——它不是最強大的，但它是唯一能夠「回應」的位置。天給了條件，地給了材料，人決定用這些條件和材料做什麼。

在天地的自然意象裡，人像是**種子**。種子沒法選擇落在哪裡（地是被給予的），沒法決定今年的雨水夠不夠（天是不可控的），但種子可以選擇發芽還是不發芽。這聽起來不像一種選擇——但在三才的邏輯裡，這是唯一真正的選擇。種子也可以選擇腐爛，也可以選擇休眠，也可以選擇在最不適合的季節拼命生長然後死掉。都是選擇。

**自然意象譜系**：

| 意象 | 含義 |
|:--|:--|
| **穀穗** | 成熟——知道自己什麼時候該低頭、什麼時候該挺直 |
| **橋** | 連接天和地——你的行動讓趨勢和積累發生關係 |
| **手** | 唯一可以向多個方向運動的肢體——指向哪裡、握住什麼、鬆開什麼 |
| **火種** | 人可以點燃，也可以熄滅——主動性是一粒很小的火，但可以燎原 |
| **蘆葦** | 在風中彎曲但不折斷——看似柔弱但能活到風停之後 |
| **弓** | 拉滿還是鬆著——你的狀態決定你蓄了多少力 |
| **羅盤** | 在天地的混沌中，人給自己定方向的能力 |
| **船槳** | 你不能決定風向，但你可以決定怎麼劃 |
| **回聲** | 你說什麼，山谷就回給你什麼——人的言行會塑造人自己的環境 |

#### 人的維度：不只「我」，還有「我與你」

三才的「人」位，最容易誤解為只講「個人」——你的選擇、你的態度、你的行動。但《周易》講的人，從來不是孤島上的人。

人在天地之間，從來不是一個人站著的。你的每一個選擇都牽動著別人，別人的每一個選擇也在定義你的處境。所以在三才框架裡，**「人」位同時包含兩個層面**：

| 層面 | 核心問題 | 說明 |
|:--|:--|:--|
| **個人的能動性** | 你要做什麼？ | 你的選擇、態度、意願、行動——這些仍然是「人」的核心，因為最終決定是你下的 |
| **人際的互動性** | 你和誰在一起？你們之間發生了什麼？ | 你的處境中涉及的人——他們的想法、你們的關係結構、互動中的張力與支持 |

這兩個層面不是分開的兩個步驟，而是同一個「人」位的兩個側面。當 AI 在 Step 2 中解讀人位時，需要同時照看這兩個層面：你不僅要問「這個人在這個場景裡選擇做什麼」，也要問「這個場景裡還有誰？他們的選擇如何與這個人的選擇交織？」

**人的化身（在人生處境中）**：

| 化身 | 自然對應 | 在處境中的樣貌 |
|:--|:--|:--|
| **選擇** | 手的朝向 | 你主動做的決定——跳還是留、說還是沉默、進還是退 |
| **態度** | 種子的狀態 | 面對處境的方式——抗爭、接受、逃避、觀望、偽裝 |
| **意願** | 火的溫度 | 你真正想要的——不是「應該的」，是你心裡那個微弱的聲音 |
| **行動** | 河流的改道 | 你實際做了什麼——不是你想的，是你做的 |
| **言行** | 漣漪 | 你說的話、你傳遞出去的東西——每一句話都在改變水面 |
| **注意力** | 光的聚焦 | 你把時間花在哪裡——現代人最真實的人位選擇 |
| **立場** | 羅盤的指向 | 你站在哪一邊、給自己劃的邊界、對外的表態 |
| **關係動態** | 水流交匯 | 你和他人的互動模式——誰主導、誰跟隨、是合作還是對抗、是依賴還是各自獨立 |
| **社會角色** | 橋的兩端 | 你在這個場景裡的身份——是員工、伴侶、父母、朋友還是領導者；角色賦予你的權力與責任 |
| **他人意志** | 多股風向 | 處境中其他人各自想要什麼？他們的意願和你的意願是交織、衝突還是各自平行 |
| **共同體** | 共生林 | 你不是單獨的個體——家庭、團隊、社群對你的期待，以及你對他們的責任感 |
| **溝通/張力** | 震動的頻率 | 話說出來了還是沒說？溝通是順暢還是卡住？沉默本身是不是一種表達？ |

**人的核心問題**：
> **「現在，你要做什麼？」**
不是「你應該做什麼」，也不是「什麼是最優的」。是——在這個天之下、地之上，你，打算做什麼。

而如果你身邊有其他人——**「你們」，打算做什麼？**

## 第二部分：以場景為中心的三才詮釋

### 核心方法：三面鏡子

三才映射不是「把場景資訊分類，放進三個盒子裡」。

而是：**同一個場景，用三種不同的鏡頭去看它。**

- **用天的鏡頭看**：看到的是趨勢、環境、那些比你大的力量——時代在什麼季節、制度在如何運轉、運氣站在哪一邊
- **用地的鏡頭看**：看到的是積累、基礎、你站著的那片土壤——你的能力有多深、你的資源有多少、你的慣性在往哪個方向拉
- **用人—不，用「人」的鏡頭看**：看到的是選擇、關係、你與他人如何共舞——你自己想做什麼，以及你身邊的人在想什麼、要什麼

---

### 映射流程：逐位置推进

Step 2 的核心节奏：**每轮只映射一个位置**。AI 完成一个位置（确认并口头小结）后，再开启下一个。

```
第一轮：地位映射 → 用户确认 → 口头小结
第二轮：人位映射 → 用户确认 → 口头小结
第三轮：天位映射 → 用户确认 → 口头小结
三轮完成后 → 输出正式的三才定位结果
```

映射顺序保持不变——**地 → 人 → 天**，从最确定的根基出发，逐步推到最不确定的环境：

1. **地** — 先看清自己的基地：我有什麼？我的根基穩不穩？我的慣性在往哪個方向拉？
2. **人** — 再看自己的選擇和身邊的人：我要做什麼？還有誰在這個場景裡？我們之間的關係是怎樣的？
3. **天** — 最後看外部環境：現在是什麼時候？什麼力量在我之上運作？趨勢在成全還是阻礙我？

---

### 逐轮方法

每轮 AI 做三件事，用概念空间的隐喻去感受场景：

1. **打开隐喻场** — 先浸泡在该位置的意象里（参考「第一部分」该位置的概念空间），选出 3-4 个与该场景最相关的意象
2. **用隐喻感受场景** — 不要分类，而是问「如果这个场景是一片土壤／一阵风／一只手的动作，那是什么质地的？」
3. **给出问题卡片** — 每个位置给 4-6 张卡片，每张 = 一个可能的诠释 + 一个直接问用户的问题。用户选一张或自述感受

### 问题卡片标准

```
**卡片 A：[一句话标题]**
[2-3 句描述，让用户感受到「这就是我的情况」]
> 👉 [直接问用户的问题]

**卡片 B：[一句话标题]**
...

最后留开口：
> 👉 或者你的理解不在上面——告诉我你的感觉。
> 👉 卡片都不准？那我再想想。
```

> 关键态度：你在邀请用户一起探索，不是考他。用户不需要懂三才——只需要感受「哪个说法让我觉得『对，我就是这样的』」。

### 「都不準」回退机制（适用于任一轮）

用户说「都不準」时，AI 在当前轮内处理，不阻塞其他位置：

**第一步：自检** — 问自己：刚才的卡片是否通过了阴阳互换测试和位置互换测试？

**第二步：重新出牌** — 换一组镜头，结合天/地/人三才之间的关系重新诠释。参考 `references/八卦表.md` 的自然意象。

**第三步：第二次仍不准** — 直接问用户开放问题：「你觉得这个位置在你的场景里更像什么？你心里已经有了一个比喻吗？」

### 第一轮：地位映射

> **AI 在这一轮只做这件事**：打开地的概念空间，给出卡片，等用户确认。
>
> 确认后口头小结：`「好的，你的地像「……」。我们看人位。」`

**核心问题：你脚下踩着的是什么？**

从「第一部分」地的概念空间中，选出与用户场景最相关的自然意象（如：土壤、岩石、河流、根系、沼泽），用它们去感受用户的根基。

**示例卡片**（以「财务 5 年，考虑跳槽运营，老公反对」的场景为例）：

> **卡片 A：深耕过的熟地，但你要换种作物**
> 你做了五年财务。那不是随便混了五年——是同一片土地上耕作了五年，财务思维就是你的土壤成分。但你想种的作物（运营）需要的养分不一样：流量思维、数据驱动、用户心理。你的地是深耕过的熟地——问题不是地好不好，是你要种的东西和你的土壤匹配吗？
> > 👉 你觉得你的「地」有多少是可以带过去的？还是你其实在准备一片全新的土壤？
>
> **卡片 B：地里长着两个人的根**
> 老公说「太冒险」——这话不只是他的态度。它是你们共同土壤的构成。你们的经济是绑在一起的，你的跳槽决策关系到两个人的生活结构。你的地不是只有你一个人的根。
> > 👉 你做的这个决定，影响的不只是你自己的地基——你知道这一点，对你的犹豫有多大影响？
>
> **卡片 C：河边走很久才下水的人**
> 投了简历但没点发送。这个悬而未决的状态本身，就是你地质的一部分。你一直都是这样的吗——先走到河边、站很久、反复试水温、然后才决定跳或不跳？还是这次特别犹豫？
> > 👉 你觉得这是你的「一贯模式」，还是这次真的不一样？
>
> **卡片 D：表面的薄土下，有更深的东西**
> 你说的「朋友内推」可能不只是机会。你的五年财务经验累积下来的人脉——朋友愿意内推你，会不会其实是你过去五年的某种积累开花了？你的地可能不只是「财务专业」，还有「业界声誉、信任关系、被看见的能力」。这些也是地，只是比较看不见。
> > 👉 你觉得除了专业技能之外，你还有哪些「看不见的积累」自己没注意到？
>
> **卡片 E：你的身体也在说话**
> 你「没敢点发送」。那个不敢，不只存在在脑子里——它可能也住在你身体的某个地方。是胃紧了一下？是胸口发闷？是睡不着的那种焦虑？你的身体也是地——它也有它的判断。
> > 👉 你的身体对这件事的反应是什么？它想告诉你什么？
>
> > 👉 或者你的理解不在上面——告诉我，你脚下的地，你自己感觉像什么？
> >
> > 👉 卡片都不准？那我再想想。

用户确认后，AI 口头小结并推进到人位。

### 第二轮：人位映射

> **AI 在这一轮只做这件事**：打开人的概念空间，给出卡片，等用户确认。
>
> 确认后口头小结：`「明白了，你的人位关键是「……」。来看天位。」`

**核心问题：现在，你要做什么？以及，你和谁在一起？**

人位同时包含两个层面：**个人的能动性**（你的选择、态度、意愿、行动）和**人际的互动性**（涉及谁、关系结构、沟通张力）。AI 在解人位时先看个人，再看关系，最后看沟通——三位一体构成完整的人位解读。

**示例卡片**（接续上述场景）：

> **卡片 A：你已经在动了，只是卡在最后一厘米**
> 你投了简历。不管最后点不点发送，你已经做了一个主动的动作。你不是在等待，你是在和自己谈判。卡在「发送」按钮前的那一厘米——那是什么？
> > 👉 如果没有任何人会知道、没有任何后果——你会点下去吗？那个「会」或「不会」，分别在说什么？
>
> **卡片 B：你踩刹车的背后，是什么样的恐惧？**
> 卡在发送前。这是一个关键的人位时刻。恐惧本身不是问题——问题是：你在怕什么？怕新领域做不来？怕老公说「我早说了」？怕自己其实不是想要这个，只是想要「逃离现在」？还是怕——万一去了之后发现，运营也没比财务好多少？
> > 👉 说出你最真实的那个恐惧——不用给任何人看，就你自己知道的那个。
>
> **卡片 C：你和老公——他在你的决策里扮演什么角色？**
> 他说「太冒险」。这不只是一句评价——它是你们关系讯号交换中的一次关键信息。是你先告诉他、他再回应的？还是你投了之后才让他知道？他的「太冒险」在你听起来是什么味道——关心？控制？还是他自己也有说不出口的恐惧？
> > 👉 他说的「冒险」——冒的到底是谁的险？是你的职业风险，还是你们共同生活的稳定性？
>
> **卡片 D：朋友内推——你和他之间的那条看不见的线**
> 朋友不只是「机会的来源」。他把你内推进一个你没有经验的领域——这个动作改变了你们的关系。如果你进了这家公司，你的表现会反过来影响他在公司内的信誉。这条隐含的责任链，也是人位的一部分。
> > 👉 你感受到这份隐含的责任了吗？它对你来说是压力，还是动力？
>
> **卡片 E：你的「朋友内推」和「老公反对」之间的张力**
> 两边都是人，他们的方向不一样。一个把你往前推，一个把你往后拉。你站在中间——这个张力本身，可能比你跟任何单一个人的关系更能说明你的处境。
> > 👉 如果抽掉这两个人——没有内推、没有反对——你自己真正想做的是什么？
>
> **卡片 F：你没说出口的那句话**
> 在整个处境里，你对老公、对朋友、对自己——有没有一句话是你想说但没说出来的？
> > 👉 那句话是什么？如果说了，会发生什么？
>
> > 👉 或者你的理解不在上面——在这个场景里，你的「人」位（你自己和那些与你有关系的人），感觉更像什么？
> >
> > 👉 卡片都不准？那我再想想。

用户确认后，AI 口头小结并推进到天位。

---

### 第三轮：天位映射

> **AI 在这一轮只做这件事**：打开天的概念空间，给出卡片，等用户确认。
>
> 确认后口头小结：`「好的，你的天像「……」。天、地、人三个位置都确认了，下面是三才定位结果。」`

**核心问题：什么在压迫或成全你？现在是什么时候？**

天是你头上那些比你大的东西——时代、制度、趋势、他人意志、命运偶然。你可以回应它，但不能假装它不存在。

**示例卡片**（接续上述场景）：

> **卡片 A：行业季节——你站在什么风里？**
> 互联网运营目前处在什么季节？是快速生长的春夏，还是已经卷成红海的秋冬？朋友内推本身说明你在这个行业里有人——但有人和有风口是两回事。
> > 👉 你感觉那个行业是在上升期，还是已经挤满了人？你对它的「季节感」是什么？
>
> **卡片 B：退路——你的天留了门，还是关了窗？**
> 如果新岗位不适应，你回财务岗的退路还在不在？市场现在的流动性——像不像你头顶的云？你能透过它看到未来的样子吗？
> > 👉 你觉得你的退路有多宽？还是你根本不想给自己留退路？
>
> **卡片 C：老公的态度，作为你头顶的一股力**
> 同样一句「太冒险」，在天的视角下看到的不是关系（那是人位），而是一股来自上方的力。它的方向指向「不动」，力度是「存在但不强烈」。它不是让你对抗或顺从的——它是你决策空间里的一个常数，你需要先搞清楚它的走向，再决定怎么和它共处。
> > 👉 如果把老公的意见单独抽出来——不当作「你们的关系」，而当作一个「客观存在的阻力」——它有多大？它会自己消失吗？
>
> **卡片 D：你有没有看到「偶然」站在你这边？**
> 朋友正好在这个时间点内推你——巧合本身也是天的一种。不是所有的好事都是你计划来的，有些是你头顶的星空正好在那个角度照到了你。
> > 👉 你觉得这件事发生在现在，是偶然，还是某种「时候到了」？
>
> **卡片 E：你不知道你不知道的事**
> 可能有更大的力量在运作但你还没看到：公司年终会不会裁员？行业有没有政策变化？你的财务同事知不知道你在投简历——这会不会影响你留任的处境？天的一部分就是「你看不到的」。
> > 👉 你觉得有没有什么「你可能没想过的变数」，其实也在影响你的处境？
>
> > 👉 或者你的理解不在上面——你感觉「天」在这个场景里，最像哪一种天气？
> >
> > 👉 卡片都不准？那我再想想。

用户确认后，三才映射全部完成。

---

> **注意**：示例卡片以「财务转运营」场景为模板。AI 针对用户的实际场景，参考这些卡片的角度和风格，生成与该场景贴合的卡片。同一条信息可能被不同位置同时捕捉（如「老公反对」既是天位的力、也是人位的关系、也是地的构成），让用户自己确认在哪位它最显著。

## 信息充分性检查

**每轮检查**（该轮用户回应后立即进行）：

| 检查项 | 说明 |
|:--|:--|
| 该位置是否有用户确认的关键描述？ | 用户至少选了一张卡片，或给出了自己的描述 |
| 描述是否有具体细节？ | 不是泛泛的「我的根基还行」，而是具象的描述 |

该位置模糊 → 在该轮内追问一轮。仍模糊 → 标记为「模糊」，继续开启下一轮。

**全部三轮后的总检查**（推进到 Step 3 前）：
- 地、人、天三个位置是否都有描述（含标记为模糊的）？
- 三个位置都至少有一轮交互 → 推进到 Step 3，模糊标记带入。
