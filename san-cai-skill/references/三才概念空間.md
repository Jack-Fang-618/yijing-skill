# 三才概念空間：12 個基礎單元

> 本文檔提煉自 `docs/八卦倫理處境/入門_陰陽爻八卦與三才概念空間.md` §6.1。
> 每個單元 = 三才位置 × 陰陽質性 × 正負發揮方向。

---

## 什麼是「正」與「負」？

在判斷正負之前，先感受「正」和「負」各自的味譜：

### 正（得當）的味道

「正」不是道德上的「好」——是「合時、適度、合乎中道」。
- 你站在正確的位置上，用正確的力道
- 不多不少——既沒有不足，也沒有過度
- 陽的正面：剛健而不僵，推力指向生長
- 陰的正面：柔順有底，承載而不潰散

正面的核心感受：**我在對的時候，用了對的方式。**

### 負（失當）的味道

「負」不是道德上的「壞」——是「過度、不及、違背時位」。
- 你站在對的位置上，但用力過猛或根本不動
- 不是你不對——是你的姿勢不對
- 陽的負面：剛硬僵化，推力變成碾壓
- 陰的負面：柔順無骨，承載變成潰散

負面的核心感受：**我的方向沒錯，但我的方式不對。**

## 天位（上爻）－環境的得當與失當

### 天陽正・公正恆常

| 維度 | 描述 |
|:--|:--|
| **一句話** | 規則透明、公平運作——你知道努力會有回報、違規會有後果 |
| **自然隱喻** | 正午的太陽——明亮、均勻、無偏私，對所有人在同一個尺度上做交代 |
| **現實的樣子** | 制度健全、獎懲分明、法律公正、上級決策可預測 |
| **核心感受** | 這個系統是可信的。我可以放心做長期計劃。 |

### 天陽負・冷酷嚴苛

| 維度 | 描述 |
|:--|:--|
| **一句話** | 規則在運行，但它忘了為何被創造——碾壓個案、不容申訴 |
| **自然隱喻** | 烈日灼人——陽光沒有消失，但它不再帶來生命，只帶來乾旱 |
| **現實的樣子** | 法條主義、績效至上的壓迫、不講人情的制度、AI 判官不考慮個案 |
| **核心感受** | 系統不在意我。我是數字，不是人。 |

### 天陰正・包容順應

| 維度 | 描述 |
|:--|:--|
| **一句話** | 環境有彈性、給你空間——你犯錯不會被立即逐出，你不一樣不需要立即對齊 |
| **自然隱喻** | 春秋細雨——不急不緩、不剛不烈，萬物自己慢慢長 |
| **現實的樣子** | 公司給了試錯空間、家人沒有逼你、制度保留彈性條款 |
| **核心感受** | 我還可以慢慢來。天沒有掐住我。 |

### 天陰負・無原則縱容

| 維度 | 描述 |
|:--|:--|
| **一句話** | 包容失了邊界——犯錯不付代價，秩序從內部潰散 |
| **自然隱喻** | 沼澤——柔軟、沒有阻力，但待久了會沉 |
| **現實的樣子** | 沒有人守規矩、做了錯事的人沒事、你開始覺得守規矩是愚蠢 |
| **核心感受** | 這個系統沒有底線。我不知道該相信什麼。 |

## 人位（中爻）－選擇的得當與失當

### 人陽正・擔當創造

| 維度 | 描述 |
|:--|:--|
| **一句話** | 你站出來了——不是為了名利，是因為你覺得「如果沒人做，我做」 |
| **自然隱喻** | 火種——很小，但你知道它可以燎原 |
| **現實的樣子** | 主動接了一個爛攤子、在沒有人說話的時候說了實話、為了對的事冒了險 |
| **核心感受** | 該我了。我來了。 |

### 人陽負・專斷冒進

| 維度 | 描述 |
|:--|:--|
| **一句話** | 「只有我能做」——向前邁步變成獨衝，擔當變成獨裁 |
| **自然隱喻** | 烈火燎原——火勢太大，燒掉了不該燒的東西 |
| **現實的樣子** | 聽不見別人的意見、把不同意當阻力、能力越強越容易掉進去的坑 |
| **核心感受** | 我最正確。其他人都是噪音。 |

### 人陰正・謙遜協同

| 維度 | 描述 |
|:--|:--|
| **一句話** | 我在，但我不擋光——把力量用在你可能看不見的支撐上 |
| **自然隱喻** | 回聲——不發出第一個聲音，但讓聲音傳得更遠 |
| **現實的樣子** | 把舞台讓給了更適合的人、默默補上所有漏洞、在幕後成全一件事 |
| **核心感受** | 此刻的價值不在被看見，在被完成。 |

### 人陰負・退縮無主見

| 維度 | 描述 |
|:--|:--|
| **一句話** | 「都行」——不是真的無所謂，是不敢選 |
| **自然隱喻** | 斷根的蘆葦——風來了彎下去，但風過了也直不起來了 |
| **現實的樣子** | 有想法但不說、怕選錯所以不選、放棄了在場的權利 |
| **核心感受** | 我不重要。我說了也不算。 |

## 地位（初爻）－根基的得當與失當

### 地陽正・穩固有原則

| 維度 | 描述 |
|:--|:--|
| **一句話** | 站得直——基本功紮實，知道什麼風值得跟、什麼是噪音 |
| **自然隱喻** | 岩石——水流過它，它不動；不是因為固執，是因為它知道自己在哪 |
| **現實的樣子** | 專業經得起考驗、有清晰的價值觀、風口來了不焦慮 |
| **核心感受** | 我知道我是誰。底牌是硬的。 |

### 地陽負・僵硬固執

| 維度 | 描述 |
|:--|:--|
| **一句話** | 站得太直了彎不了腰——原則變成教條，地基變成圍牆 |
| **自然隱喻** | 水泥地——什麼都長不出來，因為太硬了 |
| **現實的樣子** | 「我當年就是這樣學的」、拒絕學習新工具、把習慣當原則 |
| **核心感受** | 我就是這樣的人。改不了。 |

### 地陰正・承載柔韌

| 維度 | 描述 |
|:--|:--|
| **一句話** | 接得住——什麼東西落在你身上，你都接住、消化、轉化 |
| **自然隱喻** | 沃土——落葉、枯枝、雨水，全變成下一季的養分 |
| **現實的樣子** | 能容納不同意見、從失敗中學到東西、扛住壓力不散架 |
| **核心感受** | 我可以被改變，但我不會被打散。 |

### 地陰負・軟弱隨波逐流

| 維度 | 描述 |
|:--|:--|
| **一句話** | 土太鬆了——什麼都接得住，因為對什麼都沒有立場 |
| **自然隱喻** | 流沙——你越用力，沉得越快 |
| **現實的樣子** | 一再被越界、不敢說「不」、把自己的無骨當成了寬容 |
| **核心感受** | 我沒有選擇——只能讓別人決定我的形狀。 |
