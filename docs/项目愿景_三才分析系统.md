# 项目愿景：基于三才框架的处境分析系统

> 创建：2026年6月9日

---

## 一、我们从哪里出发

### 1.1 核心框架

一套基于《易经》八卦和三才的**伦理处境分析工具**，核心逻辑是三层叠合：

```
三才位置（地/人/天） × 阴阳状态（阳爻/阴爻） × 正负发挥方向（正面/负面） 
=  3 × 2 × 2 = 12 个基础概念单元
=  每个三爻卦有 2³ = 8 种解读
=  八卦合计 8 × 8 = 64 种伦理处境
```

### 1.2 和传统占卜的根本区别

传统算命：卦象 → 吉凶（你被卦象决定）  
本系统：卦象定位处境 → 正负确认 → 吉凶（你在处境中的**选择**决定结局）

同一个卦象（同样的阴阳结构）可以走向吉或凶——取决于你把每个爻位上的阴阳发挥出了正面还是反面。卦象是地形图，不是判决书。

### 1.3 和 MBTI 的类比

MBTI 用四组二分维度（E/I, S/N, T/F, J/P）交乘出 16 种人格类型——本质是 2^4 = 16 的 LAR 操作。本系统用三组二分维度（每个位置阳/阴），也是 LAR 操作，只是：
- MBTI 答的是"我是谁"（地的层面）
- 本系统答的是"我站在哪里"+"我可以往哪走"（地+人+天三层同时运作）

## 二、我们想做什么

### 2.1 最终形态

一个**交互式网站**，用户通过回答问题来完成一次处境分析。

### 2.2 用户流程

```
选择领域（事业/感情/财运/健康/家庭/学业/人际/迁移/官司/流年）
    ↓
三道三才定位题（地/人/天，每道两个选项）→ 确定卦象（8选1）
    ↓
三道正反确认题（针对该卦每个爻追问正/负）→ 确定走向（8选1）
    ↓
输出结果页：
  · 处境素描（你站在什么地形上）
  · 吉凶判断
  · 调整建议（哪个爻可以翻面、怎么翻）
  · 《周易》对照（尝试对应六十四卦相关内容）
```

### 2.3 10 个分析领域

| 编号 | 领域 | 典型问题 |
|:--|:--|:--|
| 1 | 事业/工作 | 跳槽、创业、升迁、方向选择 |
| 2 | 感情/婚姻 | 关系状态、是否合适、该不该继续 |
| 3 | 财运/金钱 | 投资、收入趋势、大额支出时机 |
| 4 | 健康/身体 | 身体状况、康复趋势、该不该手术 |
| 5 | 家庭/子女 | 亲子关系、子女教育、长辈健康 |
| 6 | 学业/功名 | 考试、升学、换专业 |
| 7 | 人际/合作 | 合作伙伴、朋友、身边的人靠不靠谱 |
| 8 | 迁移/变动 | 搬家、出国、换城市、大转型 |
| 9 | 官司/是非 | 纠纷、诉讼、名誉风险 |
| 10 | 流年/总运 | 今年整体趋势、哪个月要小心 |

### 2.4 关于 64 种处境与《周易》六十四卦的猜想

数学上，八卦（8种阴阳结构）× 每卦 8 种正负解读 = 64 种处境，恰好等于《周易》六十四卦的数目。两者是否能在内容层面上对应上——即我们的"震卦+"是否和《周易》的"震卦"在卦辞爻辞上有关联——这需要实测验证，也是下一步研究的核心任务。

## 三、已认知的关键洞见

### 3.1 阴阳不是好/坏，是两种质性

阴和阳是两种运动方向（向外/向内）的统称，不是价值标签。吉凶取决于时位——同样的刚健（阳）在需要进攻时是正面，在需要妥协时是负面。

### 3.2 阴阳在时间中互相转化

即使坚持用"好/坏"的眼睛看世界，阴阳也不固定——好不会永远好，坏不会永远坏。正午之后是黄昏，冬至之后阳气回升。它们是一个循环的时钟。

### 3.3 概念永远宽于例子

在阐述概念时，先展开隐喻的广度（天"像"什么、地"像"什么、阴"像"什么、阳"像"什么），然后再用具体例子锚定。如果先给例子，读者会把整个概念压缩成那个例子。

### 3.4 每一格必须四层糅合

描述一个爻的状态时，不能只写"根基扎实"——必须同时让读者感受到：
1. 位置的特征（地的"慢"和"重"，人的"选择权"，天的"不可控"）
2. 阴阳的气质（阳的"硬"和"刚"，阴的"柔"和"收"）
3. 正反的方向（正是铠甲/清醒/弹性，负是围墙/瘫痪/溃散）
4. 一个织在叙事里的具体画面
